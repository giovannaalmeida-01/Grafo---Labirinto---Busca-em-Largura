# Grafo---Labirinto---Busca-em-Largura
Projeto em python que implementa um solucionador de labirintos interativo utilizando o algoritmo de Busca em Largura (Breadth-First Search - BFS).
A aplicação é construída em Python com Tkinter, oferecendo uma interface gráfica (GUI) intuitiva onde o usuário pode criar, editar e resolver labirintos visualmente.

O principal objetivo é demonstrar, de forma didática e interativa, como o algoritmo BFS pode ser aplicado para encontrar o caminho mais curto entre dois pontos em um grafo representado por uma grade 2D.
